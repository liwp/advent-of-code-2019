Advent of Code 2019
===

@liwp's AoC2019 efforts.

# Usage

[Install Clojure](https://clojure.org/guides/getting_started).

Run all the solutions that I've implemented:

    $ clj -m liwp.aoc-2019

Run the tests:

    $ clj -A:test:runner
