# Advent of Code 2019

@thatismatt's attempts using various lisps.

## Run

### Elisp

```
emacs -Q --batch -l day1.el
```

### Fennel

* Install Lua
* Install Fennel - https://github.com/bakpakin/Fennel

```
fennel day2.fnl
```

### Clojure

* Install Clojure - https://clojure.org/guides/getting_started

```
clj day4.clj
```
